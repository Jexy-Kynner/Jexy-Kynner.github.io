# ciscn_2019_c_1

先查看保护机制，64 位开了 NX 保护

发现应该是用不了 ret2syscall 的

v4!=2 退出里面的小循环，=1 进入 encrypt () 函数

看到 s 和 gets (), 填充为 0x50+8
发现 puts ()->re2libc

if ( v0 >= strlen(s) )
  break;
